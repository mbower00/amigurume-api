Welcome to Amigurume API!

User

sign up user

post api.amigurume.me/user/sign-up { "email": str, "password": str, "username": str }

log in user

post api.amigurume.me/user/log-in { "username": str, "password": str }

refresh user

get api.amigurume.me/user/refresh

log out token

get api.amigurume.me/user/log-out

get all users

get api.amigurume.me/users admin

get user by id

get api.amigurume.me/user/:id admin

clearance to admin

patch api.amigurume.me/user/clearance/:id admin

clearance to user

patch api.amigurume.me/user/clearance/:id?to-user=1 admin

Products

get all products

get api.amigurume.me/products user or admin

get all types

get api.amigurume.me/product/types user or admin

get product by id

get api.amigurume.me/product/:id user or admin

add product

post api.amigurume.me/product admin { "name": str "stock": int, "description": str, "image_url": str, "type": str }

update a product

patch api.amigurume.me/product/:id admin { "name": str "stock": int, "description": str, "image_url": str, "type": str }

delete a product

delete api.amigurume.me/product/:id admin

Orders

get all orders

get api.amigurume.me/orders admin

get orders by user id

get api.amigurume.me/orders/user/:id admin

get orders for current user

get api.amigurume.me/orders/my user or admin

get order by id

get api.amigurume.me/order/:id admin

add order

post api.amigurume.me/order user or admin { "ordered_products": [ { "id": int, "quantity": int } ] }

fulfill an order

patch api.amigurume.me/order/fulfill/:id admin

unfulfill an order

patch api.amigurume.me/order/fulfill/:id?to-null=1 admin

delete an order

delete api.amigurume.me/order/:id admin
